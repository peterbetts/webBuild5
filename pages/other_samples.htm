Featured Project : Text-based adventure game engine
An incredibly basic game engine that uses the power of 2D lists to walk the player through a text-based adventure. You can bring your own story as a .csv file or enjoy the provided demo! Version 1.1 includes an updated save/load system so you never have to worry about losing your progress. I decided to feature this project because I'm proud of the ASCII art I made for the main menu. The app isn't actually deployed anywhere (nor will it be) so you'll have to live with this screenshot instead.
